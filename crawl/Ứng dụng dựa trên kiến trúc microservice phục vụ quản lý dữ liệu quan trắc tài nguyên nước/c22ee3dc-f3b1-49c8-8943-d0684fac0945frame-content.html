V kha cnh nghip v, module tnh nng qun l d liu quan trc ti nguyn nc

Sau khi xy dng xong module tnh nng, c tch hp vo h thng ti nguyn

vic ra quyt nh cho vic khai thc ti nguyn nc di t.

e) Kt lun
Sau thi gian tm hiu phn tch cc vn hin trng v d liu quan trc ti nguyn
nc di t. Lun vn tp trung vo nghin cu v a ra gii php s dng kin trc
microservice nhm xy dng cc dch v cho php tch hp vo h thng hin c, gii
quyt cc vn : (i) Tch hp, x l v lu tr d liu quan trc; (ii) Tra cu, phn tch
thng tin d liu quan trc ti nguyn nc di t. phm vi nghin cu ny tc gi
xut xy dng cc microservice gii quyt c vn c t ra bao gm: (i) Dch
v tch hp cung cp API cho php cc h thng khc tch hp d liu; (ii) Dch v x l
v lu tr gip x l d liu v cp nht vo CSDL; (iii) Dch v tm kim cung cp cc
API tra cu; v (iv) Cng thng tin cung cp giao din khai thc d liu.
Cc microservice ni trn c ci t v tch hp th nghim vo H thng gim st
ti nguyn nc ti tnh C Mau. Sau khi tch hp, cc d liu quan trc c lu tr vo
c s d liu tp trung. Giao din dashboard khai thc d liu c th hin trc quan, bao

ngm.
