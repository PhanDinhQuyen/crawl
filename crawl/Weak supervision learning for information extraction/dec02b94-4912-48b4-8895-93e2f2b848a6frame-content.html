Thesis summary
Topic: Weak Supervision Learning for Information Extraction
Author: Nguyen Hoang Long
Supervisor: Dr. Tran Viet Trung
Keywords: Information Extraction, Weak Supervision.
1. Reason for choosing the topic: I want to build a new data collection sys-
tem for the company at my work place. In the new system, there are integrated
machine learning models that help the system to automatically extract infor-
mation without having to define the structure in advance. However, training a
machine learning model to extract information requires labeled training data,
which is too expensive to label manually. This leads to the need to research
Weak Supervision learning methods to apply in building training datasets for
information extraction problems.
2. Goals of the thesis
A new data collection system that uses machine learning models to au-
tomatically extract information from web pages on the Internet needs to
be completed.
Research on the weak supervision learning method helps reduce the cost
of data labeling when building information extraction models.
3. Main content and contribution Build three machine learning models in-
cluding main content model, content classification model, information extrac-
tion mo del using training data built by weak sup ervision method to solve the
problem of extracting information from websites. Using trained models to build
an intelligent data collection system that automatically detects and extracts
information from many websites. Contribution:
Propose a new approach in solving the problem of extracting information
from HTML data for Vietnamese language;
Apply the weak supervised learning method in building training data set
for Vietnamese natural language processing problems;
Build a smart real estate data collection system for Cengroup Corpora-
tion.
4. Methodology
Weak Supervision is a training method that allows combining many different
Weak Supervision sources to create a new better labeled dataset. This reduces
the cost of data labeling.
5. Conclusion
In this thesis, I have presented the results of my research about Weak supervi-
sion Learning for Information Extraction from websites. The problems posed
for this research, including the problem of extracting information from HTML
data, the data collection system from many different websites have been solved
and completed.
