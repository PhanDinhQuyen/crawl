MASTER'S THESIS SUMMARY
Thesis title: Application of deep learning and text embedding methods for self-admitted technical
debt detection
Author: Tran Thi Dinh Class: Data Science - CH2021B
Thesis advisor: Dr. Bui Thi Mai Anh
Keyword: Self-Admitted Technical Debt, Pretrained BERT model, Graph Convolutional
Network, Software Engineering
Summary contents:
a) The reason for choosing the topic
Conduct research to develop a novel model aimed at enhancing the accuracy of
classifying self-admitted technical debt (SATD) in source code. Evaluate the
effectiveness of this model by comparing its performance to existing models used in
source code error detection. This investigation will offer insights into advancements in
the field of technical debt detection, specifically through the utilization of deep learning
models and text processing methods, showcasing the evolution and refinement of these
techniques.
b) Research Objectives, Subject, and Scope of the Study
This work attempts to fill the current research gap by incorporating the rich semantics
hidden within R source code comments and four different Java source codes by means of
pre-trained encoder language models (LMs) -a cutting-edge source code embedding
model. Then, we employ a graph convolutional network in which each node represents a
comment. Our goal is to capture not only the link among scattered sentences, but also to
learn representations for both training data and unlabeled test data by propagating label
impact through graph convolution. In addition to comparing SOTA methods in two
datasets, we built a hybrid model, combining the pre-trained LM and CNN to compare
with DebtSniffer.

Contributions:
We introduced DebtSniffer, an innovative SATD detection tool capable of
addressing the SATD problem across two major programming languages: R and
Java. DebtSniffer leverages the power of pre-trained LM models and augments
their capabilities through the incorporation of graph convolutional networks.
We conducted an extensive evaluation using real-world datasets. Our evaluation
process involved rigorous comparisons with state-of-the-art baseline models and
an alternative hybrid model, BERT-CNN. By systematically benchmarking
DebtSniffer against these models, we demonstrated its superior performance
across various SATD categories and programming languages
d) Research Methodology
- Fundamental research: Based on the survey of literature on existing approaches in
SATD detection, we address a research gap by using advanced language models to embed
the semantics of R source code comments and Java source codes. We then apply a graph
convolutional network to represent comments as nodes. Our objective is to capture
sentence relationships and learn representations for both training and unlabeled test data
through graph convolution, improving upon prior regularization techniques.
- Analytical research: To prove the effectiveness of our model, we conduct extensive
experimental evaluations on two datasets, including R and Java languages. The analysis
on experimental results shows substantial improvements in the performance of our model
compared to existing methods.
e) Conclusion
In this work, we introduced a robust and efficient approach to SATD detection,
showcasing the potential of DebtSniffer to significantly enhance the quality and
reliability of codebases in both R and Java ecosystems. The fusion of advanced code
embedding techniques with graph convolutional networks has paved the way for more
accurate and context-aware SATD identification, marking a significant step forward in
software maintenance and quality assurance.
